## 1. 공공데이터포털 (data.go.kr)

### 신청 방법

1. **https://www.data.go.kr** 접속
2. 우상단 **[회원가입]** — 개인/기관 선택, 이메일 인증
3. 원하는 데이터셋 검색 → **[활용신청]** 버튼
4. 활용목적 기재 → 심사(자동 즉시 또는 1~3 영업일)
5. 마이페이지 → 개발계정 → `serviceKey` 확인
6. 운영 전환 시 마이페이지 → **[운영계정 전환 신청]** (트래픽 상향)

> ⚠️ 키는 모든 data.go.kr API 공용. 데이터셋별로 **활용신청** 따로 필요(키는 1개).

| 99 | 일사량·일조시간 | 지상 기후 관측자료 서비스 |

## 신청 시 공통 팁

- **활용목적**: "건축 대지 분석 연구 개발 목적. 비상업용." 으로 통일하면 심사 통과 빠름.
- **서비스키 환경변수**: `.env`에 `{서비스명}_KEY` 형태로 추가. **절대 커밋 금지**.
- **키 저장 위치**: GCP Secret Manager에도 등록 (Cloud Run 배포 시 필요).
- **일일 트래픽**: 개발계정은 1,000~10,000건 제한. 운영 전 **운영계정 전환 신청** 필수.
- **data.go.kr 활용신청 거절**: 목적 불명확이 이유인 경우 대부분. 구체적 활용 시나리오 재작성 후 재신청.
- **API 형태 불확실 항목**: ⚠️ 표시된 항목은 신청 전 포털에서 샘플 응답 확인 후 진행 권장.

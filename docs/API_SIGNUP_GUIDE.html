https://www.data.go.kr
★★★ P12 ★★ P13–14 ★ P15 약 37개
▼

신청 방법

  1. https://www.data.go.kr 접속 → 우상단 [회원가입] 클릭
  2. 개인 / 기관 선택 → 이메일 인증 완료
  3. 원하는 데이터셋 검색 → [활용신청] 버튼 클릭
  4. 활용목적 기재 → 자동 즉시 승인 또는 1~3 영업일 심사
  5. [마이페이지] → 개발계정 → serviceKey 확인
  6. 운영 전환: [마이페이지] → [운영계정 전환 신청]으로 트래픽 상향
⚠️ 키는 1개 공용이지만, 데이터셋마다 개별 [활용신청]을 따로 해야 사용 가능.

신청 시 공통 팁

📝
활용목적 문구
"건축 대지 분석 연구 개발 목적. 비상업용." 으로 통일하면 심사 통과가 빠름. 거절 시 구체적 활용 시나리오 재작성 후 재신청.
🔐
키 저장
.env에 {서비스명}_KEY 형태로 추가. 절대 커밋 금지. Cloud Run 배포 시 GCP Secret Manager에도 등록 필요.
📊
트래픽 한도
개발계정은 1,000~10,000건/일 제한. 운영 전 운영계정 전환 신청 필수. data.go.kr 기준 자동 전환 메뉴 있음.
⚠️
⚠️ 형태 불확실 항목
⚠️ 표시 항목은 신청 전 포털에서 샘플 응답 확인 후 진행 권장. 일부는 래스터(GeoTIFF) 형태만 존재.
🗂️
키 1개, 신청은 여러 개
data.go.kr는 계정당 serviceKey 1개 공용이지만, 데이터셋마다 개별 활용신청을 따로 해야 실제 호출 가능.

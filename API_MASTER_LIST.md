# 터읽기 — API 마스터 리스트

> 건축 대지 분석 툴에 통합 가능한 한국 공공 데이터 API 전체 목록.
> 용도 선택 시 관련 세트만 호출하는 구조로 단계적으로 추가 예정.

**범례**
- ✅ 기존 키로 바로 가능 (KOSIS / data.go.kr / VWorld / 카카오)
- 🔑 새 키 발급 필요
- 💰 유료 또는 비용 발생 가능
- ⚠️ API 제한 있음 (부분 공개 / 지자체별 파편화 / 형태 불확실)
- ❌ 공개 API 없음 — 사용 불가 또는 대안 필요

> **⚠️ 불확실 항목 주의**: 항목 끝에 ⚠️ 또는 ❌ 표시된 것은
> 공개 API가 없거나 지자체별 파편화 / 데이터 형태 불확실한 것.
> 목록 맨 아래 [별도 정리](#api가-불확실하거나-없는-항목-별도-정리) 참조.
